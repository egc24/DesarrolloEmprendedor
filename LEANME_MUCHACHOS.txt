# DesarrolloEmprendedor
Repositorio para el desarrollo del proyecto de la materia "Desarrollo Emprendedor"

En el repo se van a encontrar con una carpet OpenCV, que es una biblioteca para manejar de manera mas simple el input de la
camara en android, y otro proyecto que contiene un ejemplo de aplicacion para poder usar la bibliteca conjuntamente con zxing 
(otra biblioteca que permite decodificar todo tipo de codigos visuales como QR, EAN13, EAN8, ISBN, etc.), para decodificar codigos QR y EAN13. La idea es modificarlo para que no solamente contenga esa funcionalidad, sino tambien toda la que corresponde al trabajo practico. 

Para poder utilizar ambos proyectos vamos a tener que usar eclipse, y el plugin ADT (Android Development Tools), ademas una vez 
instalados, habra que instalar la sdk de android 5.0.1 que la pueden descargar desde el Android SDK Manager
